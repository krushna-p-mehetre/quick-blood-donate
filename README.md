<h2>🩸 Quick Blood Donate</h2>
<p>
  Quick Blood Donate is a full-stack centralized blood donation management system that connects donors,
  hospitals, and blood banks on a single platform to improve blood availability and emergency response.
</p>
<p>
  The backend is built using Spring Boot with RESTful APIs, and the frontend is developed using React.js.
</p>
